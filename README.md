# pymavs.github.io
Python user group at UT Arlington

## Installing
To install run 

``` bash
git clone https://github.com/PyMavs/pymavs.github.io
cd pymavs.github.io
npm install
```

## Pushing Changes
To commit and push your changes run
``` bash
git add .
git commit -m "Write what you did here as a string"
git push // or pull....idk 
```
